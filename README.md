# TallerMecanico

Aplicación desarrollada para la gestión y control de folios en un taller mecánico, permitiendo registrar, consultar y dar seguimiento a servicios y reparaciones.


# Sobre el proyecto

El sistema permite organizar información relacionada con órdenes de servicio, facilitando el control de reparaciones, historial de vehículos y seguimiento de actividades dentro del taller.


# Enfoque QA

Durante el desarrollo se realizaron pruebas enfocadas en garantizar la correcta gestión de datos y flujo de información:

- Validación de captura de folios  
- Pruebas funcionales en registro y consulta de datos  
- Verificación de integridad de información almacenada  
- Detección de errores en entrada de datos  
- Validación de comportamiento ante datos incompletos  


# Casos de prueba realizados

# Registro de folios
- Validación de campos obligatorios  
- Prevención de registros duplicados  
- Manejo de datos inválidos  

# Consulta de información
- Búsqueda correcta de registros  
- Visualización de datos completos  
- Manejo de registros inexistentes  

# Edición de datos
- Actualización correcta de información  
- Persistencia de cambios en base de datos  
- Validación de datos antes de guardar  


# Ejemplos de errores detectados

- Registros duplicados por falta de validación inicial  
- Errores en captura de datos incompletos  
- Problemas en actualización de información  
- Inconsistencias en consulta de registros
